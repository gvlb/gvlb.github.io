Autoincrement with Notion API

October 12, 2021

If you’ve ever used Notion as a tracker of any kind, you’ve probably been annoyed by the fact that there is no way to prepopulate a database in any way.

More specifically, for my habit tracker, I’ve needed to manually create each entry and each time specify the current date.

Why do it every day, when you could do it once a year with one command? Thus, here’s a small CLI utility to generate Notion entries in a database with subsequent dates.

Setup

  • Create an integration from the Notion website
  • Go to your database page in Notion and click Share to give privilege to your newly created integration
  • Clone the repository
$ gh repo clone ovalb/notion-autoincrement
  • Create a file called .env in the just cloned folder and add:
    • Your NOTION_TOKEN received at the creation of your Integration
    • Your DATABASE_ID . Here’s how to find it.

Run the script and enjoy.

$ node main.js -365 #number of entries to create
